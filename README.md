## 👋 About Me

I'm a Frontend-heavy Full Stack Engineer based in Bengaluru, India, with **4 years** of experience turning messy state and real-time data problems into fast, predictable UI.

- 🏢 By day, I build enterprise **B2B telecom portals** at **Netcracker Technology** — modular Angular architecture, NgRx state, and Playwright test coverage for a live customer-facing platform.
- 🦅 By night, I'm the sole engineer behind **[Garuda](https://www.vineeth.software)** — a real-time Indian stock market analytics platform that decodes ~1,500 binary WebSocket ticks a second and renders them on live TradingView charts.
- 🧠 I think in state machines before I think in components — NgRx, RxJS, and predictable data flow are how I keep large frontends sane.
- 🛠️ I lean on **Claude Code, GitHub Copilot, Cursor, Codex, and Antigravity** daily to move faster without cutting corners.

## 🦅 Flagship Project — Garuda

<p align="center"><i>A solo-built, real-time Indian stock market analytics platform — from binary protocol decoding to a live TradingView chart, with nobody else touching the codebase.</i></p>

<div align="center">

| Metric | Result |
|---|---|
| 📡 Throughput | ~1,500 binary WebSocket packets/sec (NSE + MCX) |
| ⚡ Symbol Search | Sub-20ms across a 100K-row table (pg_trgm GIN index) |
| 🧮 Signal Engine | Proprietary demand/supply algorithm on a `Float64Array` ring buffer |
| 🔐 Auth | Zero-trust — Argon2id password hashing + Ed25519 JWT |
| 💸 Infra Cost | Under €4/month on a single Hetzner CX22 VM, <20% CPU |

</div>
